Spyfall WebApp

Grails-App of the module "Web Engineering" with Christian Ribeaud at the Fachhochschule Nordwestschweiz FHNW

GitHub Repository

Project Description

This game is the digitalisation of the card game spyfall:
All players receive cards showing the same location like casino, spa, school etc. But one player receives a card that says "Spy" instead of the location. Players then start asking each other questions. The goal is to find the spy and the spy's goal is to create good stories and to figure out the location.

Features

  • Manage Players: add, update, delete
  • Manage Locations: add, update, delete
  • Start Game: Random spy and location assignment
